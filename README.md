# 创建.net跨平台的PC端托盘程序

* 可以运行在windows上
* 可以运行在Linux桌面版上
* 启动websocket服务与桌面交互
* 通过websocket服务接收到指令调用相应dll的程序，指令为json格式
* dll程序有相同的类名/方法名/参数，通过动态方式加载，通过约定提取接口类与方法
